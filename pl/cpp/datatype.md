#Array

数组是内存中连续的一段存储空间，数组中每个元素都具有相同的类型，相同的大小空间。访问数组中的每一个元素都是O(1)的时间。

    int a[] = {1,2,3}

数组名作为右值返回数组中首个元素的内存地址。下面代码中两种方式都以进行数组元素的访问。同时在函数传参数的时候也可以传递数组的首地址给一个指针。

    int a[]  = {1,2,3};
    printf("%d\n", a[1]);
    printf("%d\n", *(a + 1));
